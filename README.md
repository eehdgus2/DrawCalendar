# DrawCalendar
 
#### 프로젝트 목적

새로운 제테크 문화가 된 한정판 신발 예약 구매 시스템을 기반으로
예약 일정 관리를 용이하게 하는 것

#### 사용 언어 및 도구(window 기반 작성)
Java

JavaScript

Jquery

HTML5

CSS

Spring

JdbcTemplate

Mybatis
